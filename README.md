# Civic AI Complaint System (NLP-based)

A web app that lets citizens register once, then submit civic complaints with photos.  
The system automatically predicts **category** and **priority**, and forwards the complaint to the right **department dashboard**.

## Features
- One-time **Register + Verify (OTP-style)** + Login
- Submit complaint with **2 proof images**
- AI-like logic: **Category + Priority prediction**
- Auto **department routing** + department dashboards
- Works on **mobile / laptop / PC** (responsive UI)

## Tech Stack
- Python 3.12
- Flask
- SQLite
- HTML/CSS (Jinja templates)
